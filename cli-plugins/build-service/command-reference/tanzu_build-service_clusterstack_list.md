## tanzu build-service clusterstack list

List cluster stacks

### Synopsis

Prints a table of the most important information about cluster-scoped stacks in the cluster.

```
tanzu build-service clusterstack list [flags]
```

### Examples

```
tanzu build-service clusterstack list
```

### Options

```
  -h, --help   help for list
```
